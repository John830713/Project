//==============================================================================
// Pet Layer - Desktop Pet Host
//==============================================================================

Purpose:
  Desktop pet window that serves as the host/carrier UI for all feature modules.
  Supports drag-drop file processing, right-click context menu with pet controls,
  window dragging, random movement, and pet-specific settings.

Contents:
  MainWindow         - Layered transparent window rendering Pet.png; supports
                       file drops, right-click context menu, random screen
                       movement, and slider popups for fine-tuning.
  PetConfig          - INI-based config (Config/Config_Pet.ini) for all pet
                       settings: position, opacity, always-on-top, movement.

Right-click menu structure:
  Pet >
    [✓] 置頂                — Toggle WS_EX_TOPMOST
    移動 >
      [●] 靜止 / [○] 移動   — Radio: enable/disable movement timer
      ---
      步伐...               — Opens slider popup (1-50 px)
      速度...               — Opens slider popup (10-1000 ms)
      [✓] 穿梭              — Toggle edge wrap vs bounce
    ---
    透明度...               — Opens slider popup (0-255)
    比例...                 — Opens slider popup (10-500 %)
  Function >
    [module items...]
  ---
  Settings...
  ---
  About Project...
  ---
  Exit

Movement system:
  - Timer-based (SetTimer ID=1) at moveSpeed interval
  - Random initial direction (dx, dy from {-1,0,1}, not both zero)
  - Each tick: window += (dx * step, dy * step)
  - Screen edges: bounce (reverse axis) or shuttle (wrap to opposite side)
  - Timer auto-stops on WM_DESTROY or when moveEnabled = false

SliderPopups:
  - Modeless popup window with Trackbar + Edit + Range label
  - Trackbar scroll updates edit and applies change via PostMessage
  - Edit accepts keyboard input; pressing Enter (IDOK) syncs to trackbar
  - Values clamped to min/max range
  - Parent disabled while popup is open
